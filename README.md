# Security Lab Portfolio

This repository contains structured security investigation reports based on hands-on labs and datasets.
Reports focus on investigative reasoning, evidence handling, and risk assessment.

## How to Navigate
Reports are organized by domain:
- Email security
- Endpoint security
- Malware analysis
- Threat intelligence

Each lab folder contains a `report.md` and supporting evidence artifacts.

## Documentation Standards
- Scenarios are summarized in my own words.
- No flags, walkthroughs, or restricted content are published.
- Conclusions are supported by evidence and include risk impact and recommendations.
